05.替换空格

May 27, 2020
1 min read
剑指offer

题目描述：

请实现一个函数，把字符串 s 中的每个空格替换成”%20”。

示例 1：

输入：s = “We are happy.” 输出：“We%20are%20happy.”

限制：

0 <= s 的长度 <= 10000

解题思路：

新建一个StringBuilder，遍历给定字符串s，如果遇到空格，就append一个“%20”，否则就直接append上当前s的字符。

Java代码：

class Solution 
{
    public String replaceSpace(String s) 
    {
        StringBuilder str = new StringBuilder();
        for(int i = 0; i < s.length(); i++)
        {
        	if(s.charAt(i) != ' ')
        		str.append(s.charAt(i));
        	else
        		str.append("%20");
        }
        return str.toString();
    }
}
