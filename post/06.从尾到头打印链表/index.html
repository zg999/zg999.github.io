06.从尾到头打印链表

May 27, 2020
1 min read
剑指offer

题目描述：

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

 

示例 1：

输入：head = [1,3,2] 输出：[2,3,1]  

限制：

0 <= 链表长度 <= 10000

解题思路：

后进先出栈，秒出。

#Java代码：

import java.util.*;

public class Solution 
{
	public int[] reversePrint(ListNode head) 
	{
		LinkedList<Integer> stack = new LinkedList<>();
		ListNode p = head;
		int len = 0;
		while(p != null)
		{
			stack.push(p.val);
			len++;
			p = p.next;
		}
		int[] ans = new int[len];
		for(int i = 0; !stack.isEmpty(); i++)
		{
			ans[i] = stack.pop();
		}
		return ans;
    }
}

class ListNode
{
	int val;
	ListNode next;
	ListNode(int x)
	{
		val = x;
	}
}
